一、实验目的
掌握使用传统机器学习方法处理图像分类任务的基本流程。
对比不同机器学习模型（KNN、朴素贝叶斯、逻辑回归、SVM、决策树、随机森林）在手写数字识别任务上的性能差异。
理解图像数据的特征表示、模型训练与评估方法，以及像素特征的局限性。

二、实验数据
本次实验使用 scikit-learn 内置的 digits 数据集，包含 1797 张 8×8 像素的手写数字图像，对应标签为 0-9 共 10 个类别。
图像总数：1797 张
图像尺寸：8×8 像素（展平为 64 维特征向量）
数据划分：按 75%:25% 的比例划分为训练集与测试

三、实验总结 
本次实验成功对比了 6 种传统机器学习模型在手写数字识别任务上的性能，结果表明 KNN 和 SVM 在该任务上表现最优，
而朴素贝叶斯和单棵决策树的表现较差。通过实验，我不仅掌握了传统机器学习处理图像的基本流程，也理解了不同模型的特性与适用场景，
以及像素特征在图像任务中的局限性。
